System Requirements

The 3.0 Java driver will run with Java 6 or later. However, specific features require Java 7:

  • By default SSL support requires Java 7 in order to perform SSL host name verification, which is enabled by default. See SSL for details on how to disable host name verification.
  • By default the asynchronous API requires Java 7, as it relies on AsynchronousSocketChannel for its implementation. See Async for details on how to use Netty instead.
