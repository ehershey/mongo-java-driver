The next generation Java driver for MongoDB

  • Synchronous library

Quick Start

Introduction

The official MongoDB Java Driver providing both synchronus and asynchronus interaction with MongoDB. Powering the drivers is a new driver core and a bson library.

Features

Bson Library
A standalone bson library, with a new Codec infrastructure that you can use to build high-performance encoders and decoders without requiring an intermediate Map instance
Core driver
A new core driver on top of which you can build alternative or experimental driver APIs
MongoDB Java Driver
An updated Java driver that includes the legacy API as well as a new generic MongoCollection interface that complies with a new cross-driver CRUD specification.
MongoDB Async Driver
A new asynchronous API that can leverage either Netty or Java 7's AsynchronousSocketChannel for fast and non blocking io.

Quick Start

The recommended way to get started using the mongodb-java-driver in your project is with a dependency management system – the snippet below can be copied and pasted into your build.

Need help? See our getting started guides on building with Maven and Gradle.

Download 

  <dependencies>
      <dependency>
          <groupId>org.mongodb</groupId>
          <artifactId>mongo-java-driver</artifactId>
          <version>3.0.0-rc1</version>
      </dependency>
  </dependencies>



    dependencies {
        compile 'org.mongodb:mongo-java-driver:3.0.0-rc1'
    }



  <dependencies>
